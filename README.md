## Getting Started

Proyecto basico generado con Vite, usando React y javascript, con npm se instalaron las librerias de @react-three/drei @react-three/fiber 

### Prerequisites

* node
  - https://nodejs.org/es
* vite
  - https://es.vitejs.dev/
* npm
  - https://www.npmjs.com/
  ```sh
  npm create vite .
  npm install dev
  npm run dev
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/suweed/configurator.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. start project
   ```npm
   npm run dev
   ```
